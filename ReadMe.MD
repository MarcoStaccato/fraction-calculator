# Fraction Calculator

Fraction Operations

## Description

You can do sum, substraction and division of fractions!

## Getting Started

### Dependencies

* At least java 8 version 
* Maven

### Building

* run `mvn package`
* A jar executable should be generated at `/target/calculator.jar`

### Executing program

* run `java -jar target/calculator.jar`

### Unit Testing 

* run `mvn test`
